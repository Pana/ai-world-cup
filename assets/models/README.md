# AI Model Icons

- Format: SVG
- Source package: `@lobehub/icons-static-svg`
- Package version: `1.91.0`
- Source repository: https://github.com/lobehub/lobe-icons
- Package license: MIT
- Downloaded: 2026-06-11
- Model mapping: `manifest.json`

The GPT entry uses the OpenAI icon. The GLM entry uses the Zhipu AI icon,
and the Qianwen entry uses the Qwen icon.

OpenAI and Grok are monochrome SVGs. When used through an `<img>` element,
they render with the SVG's default color. Use CSS filters or inline the SVG
if a different monochrome color is required.

Recommended rendering:

```css
.model-icon {
  width: 32px;
  height: 32px;
  object-fit: contain;
}
```
